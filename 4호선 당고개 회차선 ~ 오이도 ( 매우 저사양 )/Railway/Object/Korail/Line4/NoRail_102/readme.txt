///////////////////////////////////////////////////////////////////////////////////////////////////
// レール無し軌道
///////////////////////////////////////////////////////////////////////////////////////////////////

複線用の高架線だけど、現在は未着工や廃止などで使っていなかったり、レールが撤去されていたりする部分
を表現するのに使います。
Rail 構文で使えますが、0 から分岐するとポイント通過時の轍の音が鳴ってしまうので、ちょっとだけずらし
て設置して下さい。



///////////////////////////////////////////////////////////////////////////////////////////////////
// 使用方法
///////////////////////////////////////////////////////////////////////////////////////////////////

解凍したらNoRail.csv をユーザが開発している路線のオブジェクトディレクトリ（フォルダ）にセットして下
さい。その際は他のオブジェクトとごっちゃにしないよう、ディレクトリレベルで管理されることをお勧めし
ます。

.Rail 構文を使って設置出来ます。
